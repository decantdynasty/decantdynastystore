Place future optimized 3D fragrance assets in this folder.

The current hero loads `decant.gltf`, which requires `scene.bin` beside it. If
you replace the model, keep both filenames or update the path in `app.js`.
Prefer compressed `.glb` models with baked or lightweight materials for future
single-file assets.

Current atomizer model: “Atomizador para Perfume” by LEOLATO, licensed under
CC BY 4.0 and sourced from Sketchfab.
